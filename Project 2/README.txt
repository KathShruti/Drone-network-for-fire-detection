README
To use this application first run "tracker_server_TCP.py" on your IDE.
Once "tracker_server_TCP.py" is running, open two seperate command prompts.
In both command prompts change directory to the path where "p2p_client_UDP.py" is stored.
In both command prompts type "python p2p_client_UDP.py" to run the two clients.
To send data to the other client press enter in the command prompt.
